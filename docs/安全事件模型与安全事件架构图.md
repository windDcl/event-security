## 2. 自动聚合生成安全事件流程

```mermaid
flowchart TD
  A["合并告警/日志进入处理流程"] --> B{"新建还是更新?"}

  B -->|新建| C{"是否匹配已有安全事件?"}
  C -->|是| D["更新安全事件\n合入告警/日志/实体"]
  C -->|否| E{"是否命中模型入口告警?"}
  E -->|否| F["跳过处理"]
  E -->|是| G["创建安全事件"]
  G --> H["启动异步关联搜索"]
  H --> I["执行模型 DSL\nprocessList / processUnit"]
  I --> J["查询关联告警/日志/实体/证据"]
  J --> K["更新事件详情\n告警/图谱/证据/影响面"]

  B -->|更新| L{"是否已创建过该模型事件?"}
  L -->|是| M["更新安全事件字段"]
  L -->|否| E

  D --> K
  M --> K
```

## 3. 人工创建安全事件流程

```mermaid
flowchart TD
  A["安全事件列表"] --> B["点击创建安全事件"]
  B --> C["打开日志查询弹窗"]
  C --> D["选择时间范围 / 输入 HQL / 选择字段模板"]
  D --> E["查询日志或事件"]
  E --> F{"是否选择至少 1 条记录?"}
  F -->|否| G["禁用下一步或提示选择记录"]
  F -->|是| H["进入事件详情表单"]
  H --> I["填写事件名称"]
  H --> J["选择攻击场景"]
  H --> K["选择严重等级"]
  H --> L["填写责任人/解决办法"]
  I --> M{"必填项是否完整?"}
  J --> M
  K --> M
  M -->|否| N["提示补全必填项"]
  M -->|是| O["提交创建"]
  O --> P["写入安全事件索引"]
  P --> Q["绑定所选日志/事件为关联数据"]
  Q --> R["进入事件列表或详情页"]
```

## 5. 模型 DSL 执行图

```mermaid
flowchart TD
  A["入口告警命中模型"] --> B["构建变量上下文\n例如 主机IP/进程pid/文件MD5"]
  B --> C["读取模型 raw/DSL"]
  C --> D["解析 processList"]
  D --> E["替换 condition 中的变量"]
  E --> F["按 source 选择查询源\n告警/日志"]
  F --> G["按 time 计算查询窗口"]
  G --> H["查询 ES"]
  H --> I{"是否有 processUnit?"}
  I -->|是| J["递归执行子 processList"]
  J --> E
  I -->|否| K["记录关联结果"]
  K --> L["生成图谱节点和边"]
  L --> M["更新安全事件"]
```
