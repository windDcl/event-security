## 2026-06-17 第二阶段：研判能力

### 已完成

- M11 概览模块
  - 告警总量 / 核心告警 / 内网 IP / 主机统计卡片
  - 攻击阶段杀伤链可视化（7 阶段，活跃阶段高亮）
  - ATT&CK 14 阶段技战术矩阵（含命中计数）
  - 证据概览统计（威胁情报 / 恶意样本 / 可疑文件）

- M13 威胁图谱模块
  - vis-network CDN 引入（零依赖，CDN 加载）
  - 图谱可视化渲染（节点着色、形状区分、边标签、箭头方向）
  - 7 种节点类型：事件 / 告警 / 主机 / 内网IP / 外网IP / 账号 / 文件
  - 节点类型筛选（按钮切换显隐）
  - 物理布局引擎（Barnes-Hut，稳定后自动关闭）
  - 缩放、拖拽交互
  - 节点 hover tooltip（类型 + 风险等级）

- M14 相关证据模块
  - 威胁情报 / 恶意样本 / 可疑文件三类 Tab 切换
  - 威胁情报表格（指标 / 类型 / 来源 / 置信度 / 匹配时间）
  - 恶意样本表格（MD5 / 名称 / 类型 / 风险等级 / 检测时间 / 数据源）
  - 可疑文件表格（路径 / MD5 / 大小 / 风险等级 / 匹配时间）

- M15 影响面模块
  - 主机 / 内网 IP / 账号三类 Tab 切换
  - 主机表格（标识 / 名称 / IP / 操作系统 / 风险等级 / 是否失陷 / 暴露期 / 次数 / 时间）
  - 内网 IP 表格（IP / 类型 / 风险 / 组织 / 时间 / 次数）
  - 账号表格（账号 / 域 / 风险 / 是否失陷 / 活跃时间 / 次数）

- M8 表格字段配置模块
  - 字段配置弹窗 UI（勾选显隐）
  - 恢复默认字段功能
